# 店铺映射说明

## 当前映射规则

### 固定映射（在 `lingxing/seller_mapping.py` 中配置）

```python
FIXED_SHOP_MAPPINGS = {
    '110521897148377600': 'TK本土店-1店',
    '122513373670998016': 'RR-EU',
    '110521891393331200': 'TK跨境店-2店',
}
```

### 规范化规则（在 `utils/data_transform.py` 中）

- 所有 `RR-` 开头的店铺ID → 统一显示为 `RR-EU`
- 其他店铺保持原样（显示为店铺ID）

## 如何添加新店铺映射

如果需要添加新的店铺ID映射，编辑 `lingxing/seller_mapping.py` 文件：

```python
FIXED_SHOP_MAPPINGS = {
    '110521897148377600': 'TK本土店-1店',
    '122513373670998016': 'RR-EU',
    '110521891393331200': 'TK跨境店-2店',
    '新店铺ID': '新店铺名称',  # 添加新映射
}
```

修改后重新运行脚本即可。

## 关于灵星市场列表API

**重要说明**：灵星的 `/erp/sc/data/seller/allMarketplace` 接口返回的是：
- 市场/国家列表（如：美国、加拿大、英国、德国等）
- **不包含**店铺ID（sid）和店铺名称（shop_name）

因此无法通过该API获取店铺信息，只能使用固定映射。
